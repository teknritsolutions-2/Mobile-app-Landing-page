# LaunchPad Template

LaunchPad is a responsive multi-page mobile app landing template built with HTML, CSS, and JavaScript for pre-launch campaigns.

```text
template-name/
+-- assets/
¦   +-- css/
¦   ¦   +-- style.css

¦   +-- js/
¦   ¦   +-- main.js

+-- pages/
¦   +-- index.html
¦   +-- home-2.html
¦   +-- about.html
¦   +-- features.html
¦   +-- blog.html
¦   +-- blog-details.html
¦   +-- pricing.html
¦   +-- reviews.html
¦   +-- contact.html
¦   +-- login.html
+-- documentation/
+-- robots.txt
+-- sitemap.xml
+-- README.md
```

## Included

- Responsive breakpoints for mobile, tablet, desktop, and large screens
- Dark and light theme toggle with system preference detection
- RTL direction toggle across header, body, and footer
- Countdown timer, sliders, loading states, and form validation
- Unique page titles, descriptions, and semantic section structure
- Placeholder SVG image assets stored locally in `assets/images/`
